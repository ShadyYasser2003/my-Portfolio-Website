## How to Use SEO Features

### 1. Update SEO Settings
1. Go to `/admin`
2. Navigate to "SEO Settings" tab
3. Update title, description, and keywords
4. Add OG image URL
5. Save changes

### 2. Before Deploying
1. Replace `https://yourdomain.com` in:
   - `/public/robots.txt`
   - `/public/sitemap.xml`
2. Update SEO settings in dashboard
3. Upload OG image (1200x630px recommended)
4. Test on Google Search Console

### 3. After Deployment
1. Submit sitemap to Google Search Console
2. Submit sitemap to Bing Webmaster Tools
3. Monitor indexing status
4. Update content regularly

## SEO Best Practices

### Title Tag
- Keep under 60 characters
- Include main keyword
- Make it unique and compelling
- Format: "Name - Job Title | Portfolio"

### Meta Description
- Keep 150-160 characters
- Include call-to-action
- Include main keywords naturally
- Make it engaging

### Keywords
- Use 5-10 relevant keywords
- Include variations
- Don't keyword stuff
- Focus on your specialties

### OG Image
- Size: 1200x630px
- Format: JPG or PNG
- Include your photo or branding
- Shows when sharing on social media

## Google Search Console Setup

1. **Verify Ownership**
   - Go to Google Search Console
   - Add property (your domain)
   - Verify via meta tag or DNS

2. **Submit Sitemap**
   - URL: `https://yourdomain.com/sitemap.xml`
   - Monitor indexing status

3. **Monitor Performance**
   - Track clicks and impressions
   - Identify top keywords
   - Fix any errors
